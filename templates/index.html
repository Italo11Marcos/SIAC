Funcionamento

Para que o sistema funcione, é preciso submeter um arquivo zipado com os currículos.

Quando o precessamento for realizado, aparecerá um botão para download dos gráficos.

Preparar os currículos

Entre no site do lattes, clicando aqui.

Procure por Buscar currículo no canto superior direito.

Busque pelo currículo desejado.

Quando abrir a página do currículo desejado, clique no botão xml no canto superior direito.

Com os currículos devidamente baixados, compacte-os.

Atenção! Mesmo sendo somente um currículo, ele ainda deve ser compactado novamente.
